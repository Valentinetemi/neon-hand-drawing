
# Air Writer ✨🖐️

Draw in the air using your hand! This project uses **MediaPipe** for hand tracking and **OpenCV** to create a neon drawing effect on a live webcam feed.

This project was built as part of my **#100DaysOfBuildingAI** challenge.

---

## Features

- Real-time hand tracking with MediaPipe and OpenCV.  
- Draw in the air by pinching your thumb and index finger.  
- Neon glow effect for a cool visual appearance.  
- Lightweight Python implementation, easy to run.
